TV Advisor VN (Static Build for cPanel)
=====================================

1) Upload folder contents to: public_html/tvadvisor/
   - index.html
   - assets/styles.css
   - assets/app.js

2) Create folder for data and upload your JSON:
   public_html/tvadvisor/data/data.json

3) Test:
   https://YOUR-DOMAIN/tvadvisor/data/data.json
   https://YOUR-DOMAIN/tvadvisor/

Notes:
- No Node.js / build required. Pure static HTML/JS/CSS.
- Update data by replacing: public_html/tvadvisor/data/data.json
- Data schema matches the app in ChatGPT canvas.
